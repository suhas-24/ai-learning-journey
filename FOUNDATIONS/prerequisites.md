# Prerequisites

You do not need to be an expert to start this repository.

You do need a little patience, because unfamiliar words will show up quickly and that is normal.

## What You Need

- comfort opening a terminal and typing simple commands
- willingness to install tools and read error messages
- patience for debugging when something does not work on the first try
- time to practice consistently, even if the practice is small

## Helpful But Not Required

- basic high-school algebra
- familiarity with GitHub
- experience editing text files in VS Code or another editor

## What You Do Not Need Yet

- deep machine learning theory
- advanced mathematics
- prior AI job experience
- knowledge of every framework, acronym, or tool
- a full understanding of words like `LLM`, `token`, or `tokenization` before you begin

## The Most Important Skill

The single most important skill is staying with confusion long enough to resolve it.

Most of the roadmap is learnable if you keep building and debugging.

## A Gentle Rule

If you hit an unfamiliar word, do not rush past it.

Look it up in [glossary.md](./glossary.md), write down the simplest meaning you can find, and continue.
