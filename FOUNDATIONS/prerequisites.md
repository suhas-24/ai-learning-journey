# Prerequisites

You do not need to be an expert to start this repository, but a few basics will make the journey smoother.

## Required

- comfort using a terminal
- willingness to install tools and read error messages
- patience for debugging
- time to practice consistently

## Helpful But Not Required

- basic high-school algebra
- familiarity with GitHub
- experience editing text files in VS Code or another editor

## What You Do Not Need Before Starting

- deep machine learning theory
- advanced mathematics
- prior AI job experience
- knowledge of every trendy framework

## The Most Important Skill

The single most important skill is staying with confusion long enough to resolve it. Most of the roadmap is learnable if you keep building and debugging.
