# Task Brief Template

Use this template to give an agent a bounded assignment. A good brief lowers confusion before the work starts.
`Bounded` means clearly limited.

## Objective

Describe the exact change in one or two sentences.

## Owned Paths

List the only files or directories the agent may edit.

## Forbidden Paths

List the files or systems that must not be touched.

## Success Criteria

- criterion 1
- criterion 2

## Checks To Run

```bash
pytest path/to/tests -q
```

## Return Format

Return:

- files changed
- checks run
- open risks or follow-up
