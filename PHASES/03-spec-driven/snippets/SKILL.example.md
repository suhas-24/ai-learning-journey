# SKILL.md Example

A `skill` file is a reusable workflow note. Instead of repeating the same teaching or review steps in every task, you write them down once and reuse them.

## Name

Documentation Review Workflow

## Purpose

Use this skill when a learning module needs to be turned from outline notes into a beginner-usable lesson.

## Workflow

1. inspect the current file and identify missing examples
2. map the lesson into concept, worked example, failure case, and practice
3. patch the file with runnable or copyable examples
4. add a checkpoint or troubleshooting note if the concept has common confusion

## Success Signal

The file teaches the concept directly instead of only naming topics.
