# Demo Script Template

## Opening

- What problem does this system solve?
- Who is the user?

## System Walkthrough

- What happens when a request comes in?
- Which components are involved?
- Where do model calls or tools appear?

## Evidence

- Which metric or scorecard should I show?
- Which trace, log, or screenshot should I show?

## Limitation

- What does the system still struggle with?
- What safeguard or fallback exists?

## Close

- What would I build next if I had one more week?
