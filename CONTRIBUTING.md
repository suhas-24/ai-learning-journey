# Contributing

This is a personal learning repository, not a collaborative project roadmap for outside contributors. It exists to document one person's path into AI engineering as honestly as possible, including unfinished work, wrong turns, and evolving opinions.

## What Is Welcome

- Forks for your own adaptation
- Inspiration, reuse, and study
- Quiet borrowing of the structure if it helps your own learning process

## What Is Not Part Of This Repo

- External pull requests
- Feature requests for changing the direction of the learning journey
- "Polish for polish's sake" edits that make the repo prettier but less honest

## If You Fork This

- Keep the phase structure, but rewrite the notes in your own voice
- Replace the wins and stuck logs with your real experience
- Do not pretend a project is complete if it is still a plan
- Prefer build logs, test evidence, and postmortems over motivational fluff

## Working Rule For This Repository

Every note here should help with one of three things:

- understanding a concept
- shipping a project
- remembering how a real problem was solved
