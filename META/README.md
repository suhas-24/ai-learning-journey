# Meta

This directory holds internal repository operations material rather than learner-facing curriculum.

- `coordination/` contains the parallel rewrite planning and status notes that were used to restructure the curriculum into modules.
- `prompts/` contains the reusable authoring prompts that future topic drafting should start from.

You usually do not need anything here while studying. Start with [../START-HERE.md](../START-HERE.md) instead.
