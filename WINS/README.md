# Wins Log

This file exists to make progress visible. Small wins count because momentum in a long curriculum is built from repeated small completions, not rare heroic bursts.

A win is not just "I felt productive." A real win is something concrete that moved the journey forward or made the next step easier.

## How To Log Wins

Each win should record:

- the date
- the phase it belongs to
- what happened
- why it matters
- what the win unlocks next

## What Counts As A Win

- finishing a small script that actually runs
- understanding a concept well enough to explain it simply
- catching and fixing a bug instead of ignoring it
- creating a reusable note that saves future time
- setting up a tool or workflow that removes friction
- completing a weekly review honestly
